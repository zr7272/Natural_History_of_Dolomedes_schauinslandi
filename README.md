# Natural_History_of_Dolomedes_schauinslandi
Supplementary material and data described in the manuscript "Natural History of Dolomedes schauinslandi, a Range Restricted Fishing Spider from Hokoreroro | Rangatira | South East Island", submitted to New Zealand Journal of Zoology. 

Data was collected by Zita Roithmair between October 2024 and February 2025 on Hokorereoro | Rangatira | South East Island. 

csv file contains measurments of carapace width (mm) and weigth (g) of 235 Dolomedes schauinslandi individuals. Measurments were taken in the field with callipers ("Precision Measuring") and a field scale to the nearest 0.01g (Digital Scale, "Professional-Mini"). 
txt file contains link to recordings of predatory behaviour of Dolomedes schauinslandi.
